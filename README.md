# Dailog Client
Dailog(Todo toy project) Client using Next.js.
This project is currently **under development**.

## Goals and Features
- Based on Next.js
- SWR
- GraphQL
- Authentication with Google OAuth

## Getting started
1. Run [Dailog Server](https://github.com/zenato/dailog-server).
2. Run `yarn install`.
3. Run `yarn dev`.
